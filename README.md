# Welcome to SuperPermaEntityRemover8000
This is a simple C++ console app for unrecoverable removing files.
## Installation
You need g++!

Compile app:
```bash
g++ main.cpp -o <name_that_you_like>
```
Then run it!
```bash
./<name_that_you_like>
```
